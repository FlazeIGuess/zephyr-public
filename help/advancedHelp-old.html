Einleitung

Nachdem Sie Zephyr erfolgreich installiert haben, können Sie es starten! Es gibt mehrere Wege, Zephyr zu starten, einmal durch ein Icon, welches Sie auf Ihrem Desktop finden. Ein weiterer Weg ist es, Zephyr durch die Suche unten links auf Ihrem Desktop zu starten. Nachdem Sie Zephyr gestartet haben, werden Sie vom Startbildschirm begrüßt. Wenn Sie das Programm das erste Mal gestartet haben, müssen Sie Ihre Lizenzdatei einlesen. Die Lizenzdatei ist in einem ".lic" Format gespeichert! Wenn Sie keine Lizenzdatei haben, richten Sie sich an "EnSerSo"!

Mögliche Fehlermeldungen

Ungültige E-Mail, abgelaufene Lizenz, ungültige Lizenz.

Ungültige E-Mail

Dieser Fehler kann drei Ursachen haben: 1) Sie haben die E-Mail falsch geschrieben. 2) Die eingegebene E-Mail hat keine Lizenzberechtigung. 3) Bei der Lizenzvergabe ist ein Fehler aufgetreten.

Abgelaufene Lizenz

Wenn dieser Fehler aufkommt, ist die Lizenz, die Sie einspielen wollten, bereits abgelaufen.

Ungültige Lizenz

Falls dieser Fehler auftritt, senden Sie die ".lic"-Datei und schildern den Fehler an den Support.

Konverter

Hier können Sie die Datei konvertieren. Mit "*" markierte Einstellungen sind verpflichtend.

Konvertierungsmodus

Hier können Sie auswählen, ob die Datei ein Kalendarium ist. Geben Sie die Spalte für die Daten an.

Datei auswählen

Wählen Sie die Excel-Tabelle, die Sie konvertieren möchten.

Dateiname

Setzen Sie entweder einen temporären oder den Standard-Dateinamen.

Temporärer Dateiname

Hier können Sie einen temporären Namen für die Ausgangsdatei angeben.

Standard-Dateiname

Setzen Sie einen Standardnamen, der in den Einstellungen gespeichert wird.

Mandantennummer

Setzen Sie die Mandantennummer und das Datum manuell oder entnehmen Sie sie aus der Excel-Datei.

Start- & Stopblatt

Wählen Sie aus, welche Blätter in der Excel-Tabelle konvertiert werden sollen.

Konvertieren & Download

Starten Sie die Konvertierung und laden Sie die Datei herunter. Aktivieren Sie den automatischen Download in den Einstellungen.

Informations-Banner

Zeigt den aktuellen Status und eventuelle Fehler an.

Einstellungen

Bearbeiten Sie verschiedene Einstellungen. Mit "*" markierte Einstellungen sind verpflichtend.

Einstellungsmodus

Wählen Sie aus, welche Einstellungen Sie bearbeiten möchten.

Erfassungsbeleg

Bearbeiten Sie Einstellungen für Erfassungsbeleg und Kalendarium.

Ausgabe-Format

Wählen Sie zwischen ".csv" oder ".txt" als Ausgabeformat.

Trenner

Wählen Sie das Trennzeichen für die Werte, z.B. "," oder ";".

Standard-Dateinamen

Setzen Sie den Standardnamen für die Ausgabe.

Automatischer Download

Aktivieren Sie den automatischen Download der konvertierten Datei.

Mandantennummer aus Excel

Geben Sie die Koordinaten für Mandantennummer und Datum an.

Manuelle Headerzeile

Geben Sie die Zeile an, in der die Lohnarten stehen.

Automatische Headerzeilen

Die Funktion basiert auf der Zeile, in der die Beschriftung "Personalnummer" steht.

Manuelle Startzeile

Geben Sie die Startzeile für die Konvertierung an.

Automatische Startzeilen

Die Funktion basiert auf der Zeile mit der Beschriftung "Personalnummer".
